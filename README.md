# Landing Page - Augusto Seixas Corretor

Landing page profissional para Augusto Seixas - Corretor e Avaliador de Imóveis.

## 📋 Descrição

Este site funciona como uma landing page de acesso centralizado aos principais links:
- 🤖 **Hero.ia** - Ecossistema de IA do Corretor
- 🏠 **Simulação de Compra** - Simule seu financiamento imobiliário
- 💎 **Ofertas Exclusivas** - Imóveis em destaque
- 📱 **Redes Sociais** - Instagram, Facebook, WhatsApp, LinkedIn, YouTube

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Personalize os links editando o arquivo `index.html`:
   - Substitua os links `href="#"` pelos links reais
   - Atualize a imagem de perfil (substitua o placeholder)
   - Ajuste os textos conforme necessário

## 🎨 Personalização

### Alterar Links
Edite as tags `<a>` no arquivo `index.html` e substitua `href="#"` pelos seus links:

```html
<!-- Exemplo -->
<a href="https://seu-link-hero-ia.com" class="link-card featured">
```

### Alterar Imagem de Perfil
Substitua o placeholder na linha:
```html
<img src="https://via.placeholder.com/150" alt="Augusto Seixas">
```

Por sua imagem real:
```html
<img src="caminho/para/sua-foto.jpg" alt="Augusto Seixas">
```

### Alterar Cores
Edite as variáveis CSS em `styles.css` na seção `:root`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    /* ... outras cores */
}
```
